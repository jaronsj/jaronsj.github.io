Color Scheme

These are the colors that will be used for the website. The bronze color would be used mostly for the background. This is to help the website to have an earthy feel and remind the user of camping. Uranian Blue and Napels Yellow would be used to for accent and mostly on the headers. The main body text would be a black color so that it is easy to read and not distracting.
